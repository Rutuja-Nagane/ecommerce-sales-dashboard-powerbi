# Ecommerce Sales Dashboard using Power BI

## Project Overview

This project focuses on building an interactive ecommerce sales dashboard using Power BI to analyze sales performance, profitability, customer behavior, and regional business trends.

The main objective of this project was to transform raw sales data into meaningful business insights through data visualization and KPI analysis.

The dashboard helps in:
- Monitoring sales performance
- Identifying profitable categories
- Analyzing regional trends
- Evaluating customer and product performance
- Supporting business decision-making

---

# Business Objective

The primary objectives of this project were to:

- Analyze ecommerce sales performance
- Track profit and revenue trends
- Identify top-performing categories and products
- Generate business insights using visual analytics
- Create an interactive dashboard for decision-making

---

# Dashboard Purpose

The dashboard was designed to provide:
- A centralized view of sales and profit performance
- Interactive business insights
- Trend analysis across categories and regions
- KPI monitoring for business growth analysis

## 1. Customer_Order_Data

## 2. Sales_Detail_Data
This dataset contains:
- Product details

- Category information
- Profit and pricing details

# Dashboard Features

The dashboard includes:

- KPI cards
- Interactive filters and slicers
- Sales trend analysis
- Profit analysis
- Category-wise insights
- Regional sales analysis
- Customer and product performance tracking

---

# KPIs Included

The dashboard tracks important business KPIs such as:

- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value
- Profit Margin
- Category-wise Revenue
- Regional Sales Performance

---

# Business Insights Generated

Some key insights identified from the dashboard:

- Certain product categories generated the highest revenue.
- Some states showed lower profitability despite high sales.
- High discounts negatively impacted profit margins.
- Seasonal sales trends were observed across different periods.
- Top-performing customers contributed significantly to overall sales.
